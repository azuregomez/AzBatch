# File-based Batch processing with Azure Batch
<h3>Business Case:<h3>
Enterprises often have file-based, IO intensive batch processes that can move to the cloud and scale horizontally with minimum code changes by leveraging Azure Batch.
https://docs.microsoft.com/en-us/azure/batch/
Azure Batch has a job scheduling mechanism that supports job dependencies:
https://docs.microsoft.com/en-us/azure/batch/batch-task-dependencies
